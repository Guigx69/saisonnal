# Saisonnal

Application Expo Router pour consulter les fruits et legumes de saison, gerer des favoris locaux et personnaliser quelques preferences sans compte.

## Commandes

```bash
npm install
npm run start
npm run lint
npx tsc --noEmit
```

## Structure actuelle

- `app/` routes Expo Router
- `components/` composants reutilisables
- `constants/` tokens de theme
- `src/data/` donnees produit locales
- `src/storage/` persistance AsyncStorage
- `src/types/` types partages
